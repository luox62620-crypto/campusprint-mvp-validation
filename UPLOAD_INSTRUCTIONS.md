# Upload Instructions

Target repository: https://github.com/luox62620-crypto/campusprint-mvp-validation

## Option A: GitHub web upload

1. Go to GitHub and create a new repository named `campusprint-mvp-validation`.
2. Keep it public or private depending on your course requirement.
3. Upload all files and folders from this package.
4. Copy the repository URL.
5. Paste the URL into Project 2 Part G and Appendix G.

## Option B: Command line push

```bash
cd campusprint-mvp-validation-repo
git remote add origin https://github.com/YOUR_USERNAME/campusprint-mvp-validation.git
git branch -M main
git push -u origin main
```

For GitLab, replace the remote URL with your GitLab repository URL.
